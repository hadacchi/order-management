# ECサイト配達状況管理スクリプト (GAS)

このプロジェクトは、Google Spreadsheet と Google Apps Script (GAS) を活用して、ECサイトでの注文および配達状況を自動管理するツールです。

## 概要

Gmail 上の特定のラベルが付与された未読メールを自動的に読み取り、メール本文を解析して配達状況（注文受付、発送済み、配達完了など）をスプレッドシートに記録・更新します。

## 機能

- **Gmail連携**: 指定したラベルの未読メールを定期的に取得。
- **自動パース**: メール内容から注文情報や配送ステータスを抽出。
- **進捗管理**: スプレッドシート上で各注文の配送状況を一元管理。

## 使い方

1. **スプレッドシートの準備**
   - 新規または既存の Google スプレッドシートを開きます。
   - メニューから「拡張機能」 > 「Apps Script」を選択します。

2. **スクリプトの配置**
   - エディタが開いたら、このリポジトリにある `.gs` ファイル（`Main.gs`, `Amazon.gs`, `Rakuten.gs` など）の内容を、それぞれのファイル名で作成してコピペします。
   - 必要に応じて `Main.gs` 内のラベル設定などを環境に合わせて変更して保存します。

3. **トリガーの設定**
   - 左側のメニューから「トリガー（時計アイコン）」を選択します。
   - 「トリガーを追加」ボタンをクリックします。
   - 実行する関数として `main` を選択し、イベントのソースを「時間主導型」、タイプを「1時間おき」などに設定して保存します。
