# ECサイト配達状況管理スクリプト (GAS)

このプロジェクトは、Google Spreadsheet と Google Apps Script (GAS) を活用して、ECサイトでの注文および配達状況を自動管理するツールです。

## 概要

Gmail 上の特定のラベルが付与された未読メールを自動的に読み取り、メール本文を解析して配達状況（注文受付、発送済み、配達完了など）をスプレッドシートに記録・更新します。

## 機能

- **Gmail連携**: 指定したラベルの未読メールを定期的に取得。
- **自動パース**: メール内容から注文情報や配送ステータスを抽出。
- **進捗管理**: スプレッドシート上で各注文の配送状況を一元管理。
